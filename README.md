# 📝 Learning-Todo-Flask
> **🌐 Live Demo:** (https://to-do-list-webapp-jcej.onrender.com)
A lightweight web application built to practice the fundamentals of **CRUD** (Create, Read, Update, Delete) operations using Python and the Flask framework.

## 🚀 The "Why" behind this project
I built this as a second project to move beyond simple text manipulation and understand how a server manages a "list" of data that changes based on user input. 

## 🛠️ Current Tech Stack
- **Language:** Python 3.x
- **Framework:** Flask (Backend)
- **Frontend:** HTML5 & CSS3
- **State Management:** Python Lists (Memory-based)

## 🚧 Work in Progress (Development Roadmap)
The project follows an iterative development cycle. I am currently learning how to bridge the gap between temporary memory and permanent storage.
- [x] **Phase 1:** Basic UI and "Add Task" POST request functionality.
- [x] **Phase 2:** Real-time list rendering using Jinja2 templates.
- [ ] **Phase 3:** Implement a "Delete" function to handle specific item indices.
- [ ] **Phase 4:** Integrate JSON File I/O for data persistence (Saving tasks).
- [ ] **Phase 5:** UI/UX Polish: Add Dark Mode and task filtering to filter finished and unfinished tasks.

## 💻 Running the Project
1. Clone the repository.
2. Ensure Flask is installed: `pip install flask`.
3. Run `python app.py` and navigate to `localhost:5000`.
